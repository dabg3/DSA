# Data structures and algorithms

A deep dive in C, CMake while improving DSA knowledge

## Build

Configure 
```shell
cmake -B build
```

Compile
```shell
cd build
make 
```

Run tests
```shell
./test_runner
```

## Project layout 

* `include` - headers
* `src`     - implementations
* `tests`   - test suites
* `unity`   - test framework
